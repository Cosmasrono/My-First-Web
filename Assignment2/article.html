HTML

This is an hypertext markup language for web pages.It describes the structure of a web page. It consists of a series of elements which tells the browser how to display content.It contains elements such as a heading tag, paragraph tag among others.

CSS

Cascading Style Sheets, is a programming language which is used to describe how HTML elements are to be displayed either on screen,paper or other media.

JavaScript

Its the world`s most famous programming language for the web. It programs the behaviors of the web pages.

Java

Its a basic programming language used for mobile,web,desktop applications. It works on different platforms such as Windows, Mac,Linux among others.
